# pixiv-crawler
A Pixiv crawler that can filter images by the numbers of bookmarks.

# Requirements
* Python 3
* Requests
* BeautifulSoup 4
```
pip3 install requests
pip3 install beautifulsoup4
pip3 install lxml
```
If socks5 proxy is used.
```
pip3 install requests[socks]
```

# Usage
```
python3 get_image.py
```
